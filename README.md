# BBC News LLM Analyzer

This project is my first experiment learning **Large Language Models (LLMs)**.

Dataset: BBC News Dataset from Kaggle.

The notebook demonstrates:

- Downloading dataset using kagglehub
- Preprocessing news articles
- Summarizing articles using FLAN-T5
- Find keywords based on articles

## Run Project

Install dependencies:

uv pip install transformers torch sentencepiece
